Willkommen bei deiner eigenen Todo Liste

Für deine Übersicht klicke auf Overview

  • Gehe zu http://127.0.0.1:5000/todo-list
  • Sende eine Post Request mit dem Namen der Liste
  • Gehe zu http://127.0.0.1:5000/todolist/list_id
  • Sende eine Get Request mit der Nummer der Liste als list_id
  • Als Beispiel: http://127.0.0.1:5000/todolist/1318d3d1-d979-47e1-a225-dab1751dbe75
  • Mit einem Delete Request kann die gewählte Liste auch gelöscht werden
  • Gehe zu http://127.0.0.1:5000/todo-list/list_id/entry/entry_id
  • Sende eine Get Request mit der Nummer der Liste als list_id und die entry_id zum ermitteln des Entrys
  • Als Beispiel: http://127.0.0.1:5000/todolist/1318d3d1-d979-47e1-a225-dab1751dbe75/entry/17983c31-2098-46e4-85d2-2934681fd8f6
